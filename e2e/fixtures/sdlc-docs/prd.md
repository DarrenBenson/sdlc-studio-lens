## 1. Project Overview

### Product Name
Widget Platform

### Purpose
A platform for managing widgets across multiple environments, providing CRUD operations and analytics.

## 2. Requirements

### Functional Requirements
- Users can create, read, update, and delete widgets
- Widgets have names, types, and status fields
- Dashboard shows widget statistics
